Developed a socially interactive web-app with the given datasets: class_query.csv and class_operation.csv

Analysis & Findings

* Finding 1: Main intention of all users is "Knowledge Seeking" and Count is 285. This implies that most of the people query in StackOverFlow for Knowledge Seeking purpose
* Finding 2: Min intention by all users is "Method Learning" and Count is 92. This implies that most of the people do not query for method learning.
* Finding 3: Max operation by all users is "scroll_down" and Count is 443067. This implies that maximum users scroll_down in stackoverflow. Scroll Down can be to read a page or to skim through.
* Finding 4: Min operation by all users is "select" and Count is 905. Least number of users select the text. Users select a text only when they need to copy the text or to search a similar question in stackoverflow or other websites.
* Finding 5: Found the main intention of each user. This can found when navigating to class_query.
* Finding 6: Found the least intention of each user. This can found when navigating to class_query.
* Finding 7: Found the maximum intention of each user. This can found when navigating to class_operation.
* Finding 8: Found the least operation of each user. This can found when navigating to class_operation.
